# Interpreter

**Interpreter** — это учебный проект интерпретатора простого программного языка, написанный на **Java** с использованием **ANTLR-грамматики**.  
Проект включает исходный код интерпретатора, грамматику языка, пример тестовых программ и скрипты для сборки и запуска.

## Содержание репозитория
- Грамматика языка: `MyGram.g4`
- Интерпретатор: `Interpriter.java`
- Скрипты
  1. `builder.sh`: Скрипт сборки проекта 
  2. `clear.sh`: Скрипт очистки временных файлов 
  3. `prog_test.sh`: Скрипт для запуска тестов интерпретатора
- Программы на языке: `*.prog`

## Требования

Чтобы собрать и запустить проект, вам понадобится:

- **Java JDK 8+**
- **ANTLR 4**
- Bash-совместимая оболочка
